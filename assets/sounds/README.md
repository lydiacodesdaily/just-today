# Audio Assets

Place audio files here:

- `tick.mp3` - Looping ticking sound for active tasks
- `chime.mp3` - Chime sound for notifications

## Recommendations

- **tick.mp3**: Short (1-2 second) subtle ticking or metronome sound
- **chime.mp3**: Gentle notification sound (soft bell or chime)

You can find royalty-free sounds at:
- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)

For testing, you can use any MP3 files temporarily.
